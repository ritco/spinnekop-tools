**2. Vervang de hardcoded APP_VERSION definitie:**

**3. Maak de title bar dynamisch in de App class:**

In App.__init__, verwijder de statische `self.title(APP_TITLE)` aanroep en vervang door:

```python
# Titel instellen met versie en omgeving (dynamisch bijgewerkt)

```

Voeg een nieuwe methode toe aan de App class:
```python
def _update_title(self, omgeving: str):
    """Update de title bar met versienummer en actieve omgeving."""
    omgeving_label = "Speelomgeving" if omgeving == "speel" else "Live (productie)"
    self.title(f"BOM Import Tool v{APP_VERSION} [{omgeving_label}]")
```

Pas _load_file aan zodat die de title updatet na omgeving-keuze:
In de _load_file methode, voeg toe na het aanroepen van self._show_analysis():
```python
self._update_title(omgeving)
```

Pas _show_start aan zodat de title terugkeert naar de geselecteerde omgeving:
In de _show_start methode, voeg toe (lees de omgeving uit start_frame):
```python
try:
    self._update_title(self.start_frame.omgeving_var.get())
except Exception:
    self._update_title("speel")
```

Pas de StartFrame.__init__ aan zodat de App titel bijgewerkt wordt zodra de radiobutton verandert. Voeg na de definitie van self.omgeving_var een trace toe. Maar StartFrame heeft geen referentie naar App. Gebruik daarom een andere aanpak: geef een callback mee via on_file_selected of via een aparte on_env_changed callback.

**Elegantere aanpak (Claude's discretie):** Voeg een `on_env_changed` parameter toe aan StartFrame.__init__ (naast on_file_selected). Pas App.__init__ aan om deze callback door te geven:

In App.__init__:
```python
self.start_frame = StartFrame(
    self,
    on_file_selected=self._load_file,
    on_env_changed=self._update_title,
)
```

In StartFrame.__init__ (voeg parameter toe en registreer trace):
```python
def __init__(self, master, on_file_selected, on_env_changed=None):
    ...
    self.on_env_changed = on_env_changed
    ...
    self.omgeving_var = ctk.StringVar(value="speel")
    if on_env_changed:
        self.omgeving_var.trace_add("write", lambda *_: on_env_changed(self.omgeving_var.get()))
```

Resultaat: elke keer dat de gebruiker van Speelomgeving naar Live wisselt (of vice versa), updatet de title bar direct.
